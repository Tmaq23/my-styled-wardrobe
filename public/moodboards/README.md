This directory contains curated fashion mood board images for each body shape type.

## Required Images

Please create collages from the fashion items and save them with these exact filenames:

**Key Features**: Fitted silhouettes, belted pieces, wrap styles, defined waist

## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: Minimum 800px width x 600px height
- **Aspect Ratio**: 4:3 or 16:9 works best
- **File Size**: Keep under 500KB for optimal loading
- **Style**: Clean collage layout with white or neutral background

## How to Create the Collages

1. Use the fashion item images provided
2. Arrange them in a grid (2-3 items per row)
3. Ensure good spacing between items
4. Add subtle shadows or borders for separation
5. Export as high-quality JPG
